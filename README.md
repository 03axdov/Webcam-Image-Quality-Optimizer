# Webcam-Resolution-Optimizer
A model for improving the quality of images, made light enough to improve webcam quality.

Based on the 'ESRGAN: Enhanced Super-Resolution Generative Adversial Networks' paper by Xintao Wang, Ke Yu, Shixiang Wu, Jinjin Gu, Yihao Liu, Chao Dong2, Chen Change Loy, Yu Qiao, Xiaoou Tang
